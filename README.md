## 🤖 AI & LLM (人工智能与大模型)

* **[system-prompts-and-models-of-ai-tools](https://github.com/x1xhlol/system-prompts-and-models-of-ai-tools)** - 汇集了 Claude, Gemini, Devin 等各类 AI 工具的 System Prompts（系统提示词）及模型信息。
* **[vibe-coding-cn](https://github.com/2025Emma/vibe-coding-cn)** - Vibe Coding 理念的中文实践与资源集合。

* **[whisper](https://github.com/openai/whisper)** - OpenAI 开源的鲁棒性极强的语音识别系统（转录与翻译）。
* **[VideoCaptioner](https://github.com/WEIFENG2333/VideoCaptioner)** - 基于 LLM 的智能视频字幕生成、断句、校正与翻译全流程工具。
* **[MinerU](https://github.com/opendatalab/MinerU)** - 将 PDF 等复杂文档转换为 LLM 可读的 Markdown/JSON 格式的智能工具。
* **[PDFMathTranslate](https://github.com/PDFMathTranslate/PDFMathTranslate)** - 保留原文档排版的 PDF 科学论文全文双语翻译工具。
* **[Professor-Synapse](https://github.com/ProfSynapse/Professor-Synapse)** - 一个旨在通过角色扮演提升 AI 交互效率的 Prompt 工程项目。
* **[Awesome-GPTs](https://github.com/lxfater/Awesome-GPTs)** - 收集了 1000+ GPTs 及 80+ 泄露的 Prompt 提示词资源。
* **[awesome-chatgpt-prompts-zh](https://github.com/PlexPt/awesome-chatgpt-prompts-zh)** - ChatGPT 中文调教与场景使用指南。
* **[Awesome-ChatGPT-Prompts-CN](https://github.com/wikieden/Awesome-ChatGPT-Prompts-CN)** - 包含咒语指南、聊天提示词及学习指南的 ChatGPT 资源库。
* **[DeOldify](https://github.com/jantic/DeOldify)** - 基于深度学习的旧照片/视频着色与修复项目。
* **[animegan2-pytorch](https://github.com/bryandlee/animegan2-pytorch)** - 将风景照片转换为动漫风格的 PyTorch 实现。
* **[LaTeX-OCR](https://github.com/lukas-blecher/LaTeX-OCR)** - 使用 ViT 将数学公式截图转换为 LaTeX 代码。
* **[AnimatedDrawings](https://github.com/facebookresearch/AnimatedDrawings)** - 让儿童手绘的人物画动起来的 AI 项目。

* **[zotero-better-notes](https://github.com/windingwind/zotero-better-notes)** - 极大地增强 Zotero 的笔记管理功能，实现“文献即知识库”。

* **[papis](https://github.com/papis/papis)** - 强大且高度可扩展的命令行文献管理器。
* **[sm18-lazy-package](https://github.com/L-M-Sherlock/sm18-lazy-package)** - SuperMemo 18 的中文懒人整合包，辅助间隔重复记忆。
* **[import-from-notion-to-joplin](https://github.com/adamfoerster/import-from-notion-to-joplin)** - 将 Notion 笔记导入 Joplin 的脚本。
